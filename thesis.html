    • Techonlogies used:

  • React to build the front-end application.
  • Python to process data and build the machine learning algorithm
  • Flask environment to house the prediction model and send predictions to the front end via API
  • Php to handle communication between the font-end and database.
  • MySQL to act as the database and store application data.

    • Description:

    This thesis project aims to create an application for veterinary doctors. Taking a dataset of colic horses, a prediction model was trained to output their survival probability.

    It was developed in collaboration with the client ‘Animal Manager ltd’. The outcome of the project was a success. All design requirements were achieved and the client was very happy with the ultimate outcome of the developed software.
